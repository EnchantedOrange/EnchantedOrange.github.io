Команды бота

Обязательно
Опционально

Для всех пользователей

!bc
Выбрать новое имя для сэра Бенедикта Камбербэтча
!снежок имя_пользователя
Бросить снежок в случайного пользователя из чата, либо пользователя, указанного после команды
!рим 1-3999
Перевод арабских чисел в римские
!кпд
Количество людей в чате делённое на количество фолловеров в процентах
!punto ghbdtn
Перевод из английской(латинской) раскладки в русскую(кириллическую) и наоборот
!wasted fudge you cabron
Потраченный переводчик с английского на русский
!реверс тевирп
Возвращает "отзеркаленный" текст, введённый после команды
!count имя_пользователя
Возвращает количество написанных Вами сообщений, либо количество сообщений, написанных пользователем, указанным после команды
!top
Возвращает топ 5 пользователей по количеству написанных сообщений
!сэм sample text
Попробуй найти письмо
!slap пользователь
Издевайся над пользователями
!соурплс
Танец в чате
!help или !команды
Список команд бота (тот самый, в котором Вы сейчас находитесь)
!дуэль пользователь
Вызвать пользователя на дуэль
!принять
Принять дуэль
!отказ
Отказаться от дуэли
!throw
Бросить кости

Для стримера

!скажи привет
Возвращает текст, написанный после команды
!опрос вариант_1 вариант_2 вариант_N
Создаёт опрос с вариантами, указанными через пробел после названия
!результат
Получение результатов опроса

Для создателя

!скажив канал привет
Написать текст в указанный канал
!host канал
Заставить бота захостить указанный канал
!delete канал
Удалить таблицу канала из базы данных
!rename старое_название новое_название
Переименовать канал в базе данных
!renameuser старое_имя новое_имя
Переименовать пользователя
!clear
Очистка базы данных
!msgv пользователь количество
Изменить количество сообщений у пользователя. "+1" добавляет сообщение, "-1" отнимает
!deleteuser пользователь
Удалить пользователя из базы данных
!123
Промотреть упоминания
!off
Выключить бота
